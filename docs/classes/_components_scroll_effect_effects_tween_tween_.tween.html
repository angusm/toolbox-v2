Class Tween

Effect for ScrollEffect that tweens CSS properties as the user scrolls.

The intent is to create an effect that can replicate a CSS animation that plays according to the scroll position rather than according to time.

    • Defined in components/scroll-effect/effects/tween/tween.ts:27

    Creates a new instance of the tween effect.

    Parameters

    • keyframes: TKeyframesConfig

      The frames which should be tweened between.

      Values are provided as an array of tuples containing a number and string defining CSS styles.

      Ex. [ [0, "opacity: 0"], [.5, "opacity: 0.5; background: blue"], [1, "opacity: 1; background: red"], ]

    • Default value __namedParameters: object = defaultOptions
      • keyframeStyle: KeyframeStyle

        Defines how keyframe positions should be treated.

        Receives values from KeyFrameStyle, can be either PIXEL or PERCENT.

        If the value is PIXEL then keyframe positions are treated as referring to absolute scroll values at which they should take place.

        If the value is PERCENT then keyframe positions are treated as referring to the scroll distance at that percentage of the valid scroll range.

        Assuming a standard GetDistanceFn that returns pixels, and the ScrollEffect set to apply to a range between 0 and 200. A keyframe at position 15 would apply after scrolling 15 pixels using the PIXEL key frame style, and at 30 pixels using the PERCENT key frame style.

        If no value is provided, this defaults to KeyframeStyle.PERCENT

      • styleTarget: HTMLElement

        The HTML element to apply tween styles to.

        If no value is provided this defaults to the ScrollEffect target.

    Returns Tween
